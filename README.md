# portal-cn-lab

一个用于归档和发布多份独立 HTML 页面的实验室仓库。  
本仓库不针对任何特定域名或网站，仅作为页面资源的集中存放与展示空间。

## 项目简介

portal-cn-lab 旨在提供一个轻量级的 HTML 页面集合，供学习、参考或快速预览使用。  
所有页面均为独立文件，无需后端支持，可直接在浏览器中打开。

## 目录说明

```
.
├── index.html         # 仓库首页 / 入口页面
├── pages/             # 存放各个独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/            # 页面引用的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md          # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，命名清晰，便于识别。
- `assets/`：存放页面所需的共用或专用资源，保持目录整洁。

## 页面归档说明

本仓库收录的页面均为静态 HTML，不包含后端逻辑或数据库依赖。  
页面内容可能涉及技术演示、样式测试、信息展示等通用用途。  
所有页面均遵循以下原则：

- 页面之间相互独立，不共享状态。
- 不包含任何指向外部特定域名的链接或资源。
- 鼓励使用相对路径引用本地资源。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面。
- 请确保新页面不与已有页面重复，且内容合规。
- 页面文件请放入 `pages/` 目录，并保持命名风格一致。
- 如有共用资源，请放入 `assets/` 对应子目录。
- 不建议在页面中包含跟踪脚本、广告或恶意代码。
